O operador `*=` multiplica uma variável por um número.

```js
myVar = myVar * 5;
```

multiplicará `myVar` por `5`. Essa expressão pode ser reescrita assim:

```js
myVar *= 5;
```

Outros exemplos

```js
let multi1 = 2;
let multi2 = 300;
let multi3 = 40;

multi1 *= 4;
multi2 *= 2;
multi3 *= 12;
```
