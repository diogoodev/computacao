Podemos facilmente incrementar ou adicionar 1 à variável com o operador ++.

`i++;`

é o equivalente a

`i = i + 1;`

Quando usamos a primeira opção eliminamos a necessidade do operador de atribuição.

Exemplo: 
```
let centena = 100

centena = centana + 1;

ou

centana++;

```
