Assim como o operador += soma, o  -= subtrai um número de uma variável.

```js
myVar = myVar - 5;

//vai subtrair 5 de myVar. Essa expressão pode ser reescrita assim:

myVar -= 5;
```

Outros exemplos:

```js
let num1 = 5;
let num2 = 10;
let num3 = 100;

num1 -= 2;
num2 -= 8;
num3 -= 20;
```
